# NLP Ranking

NLP Ranking site.


## Data Format

[`bib_map.txt`](dat/bib_map.txt)

* `0`: conference/journal ID.
* `1`: weight.
* `2`: description.

[`email_map.tsv`](dat/email_map.tsv)

* `0`: publication ID.
* `1`: author count.
* `2`: list of authors' email addresses delimited by `;`.
* `3`: list of (institute's domain name, count) pairs delimited by `;`.
